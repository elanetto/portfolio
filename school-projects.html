"Tell me, Harry. What exactly is the function of a rubber duck?"
- Arthur Weasley.

In programming, 'rubber ducking' is a problem-solving technique, where the programmer asks a rubber duck what to do, while explaining the error. Often, doing this will result the programmer to get a new perspective, and then being able to fix the problem.

I have learned this fun and creative technique at Noroff, where I am currently studying how to code using:

CSS HTML JavaScript
